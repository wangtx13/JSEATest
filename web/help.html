Help

  1. JSEA can help you to understand source code more easily and quickly by generating some topics with the help of Mallet.
  2. What you need to do:
    • Upload the directory of source code that need to process.
    • Select preprocessing steps as your demands, and we recommend do all steps.
    • Click “Start Preprocessing”.
    • Open bash and enter “MALLET” directory.
    • Run “topic-modeling.sh”: bash topic-modeling.sh -input_file -topic_number
      • -input_file: the project name, like JHotDraw.
      • -topic_number: the number of topics.
    • You can use the system via “show” page (The project overview page) and “search” page (JSEA-Search).
